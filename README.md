# Minhas Contas

Sistema de controle financeiro pessoal para gerenciamento de contas, despesas, receitas e planejamentos mensais.

## 📋 Funcionalidades Planejadas

- **Cadastro de Contas:** Registro de contas bancárias, cartões de crédito e carteira.
- **Transações:** Lançamento de despesas, receitas e transferências entre contas.
- **Categorização:** Organização por categorias (Ex: Alimentação, Moradia, Transporte, Lazer).
- **Orçamentos:** Definição de limites de gastos por categoria.
- **Relatórios:** Gráficos e resumos visuais de receitas e despesas.

## 🛠️ Tecnologias Recomendadas

- **Frontend:** HTML, CSS moderno, JavaScript (ou framework como React / Next.js)
- **Backend/Banco de Dados:** Supabase / PostgreSQL ou LocalStorage para início rápido
